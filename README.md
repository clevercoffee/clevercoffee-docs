# About
<p>This repository contains the documentation for the CleverCoffee PID controller project.<br>
You can find our project website here: [Clever Coffee Website](https://clevercoffee.de).

# Dependencies
<p>This documentation is build with [Material for MkDocs](https://squidfunk.github.io/mkdocs-material/).<br>
Multilanguage support is based on [mkdocs-static-i18n](https://github.com/ultrabug/mkdocs-static-i18/).<br>
Different versions of this documentation are deployed with [mike](https://github.com/jimporter/mike)
